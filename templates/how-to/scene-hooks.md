# Scene Hooks

A hook is a situation a player can grab. It is not a plot command.

Strong hooks usually include:

- a named person, place, or object
- a visible pressure
- a choice the party can make now
- a consequence if ignored
- one concrete sensory or behavioral detail

Weak hooks are vague: trouble in town, rumors of danger, a mysterious stranger.
Stronger hooks are specific: the kite-rack foreman is burning route tags while
the Accord clerk pretends not to see.

When creating a hook, leave the answer open. The hook invites action; it does
not decide the result.
