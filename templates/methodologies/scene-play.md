# Scene Play

Compatibility index. Generated TURN_START files link directly to one required
methodology for the actor's role and turn type.

- DM turns use [`scene-play-dm.md`](scene-play-dm.md).
- Player turns use [`scene-play-player.md`](scene-play-player.md).
- DM scene-boundary turns use [`scene-transition-dm.md`](scene-transition-dm.md).
- Player cleanup turns between scenes use
  [`scene-housekeeping-player.md`](scene-housekeeping-player.md).
- Player rapid-response turns use
  [`rapid-response-player.md`](rapid-response-player.md).
