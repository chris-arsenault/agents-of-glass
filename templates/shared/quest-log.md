# Quest Log

**Stub.** The DM owns and updates this document.

A durable narrative quest log — what the party is pursuing, what they've completed, what hooks are open. Prose, not a checklist. Encyclopedia-shaped not enforced; this can be more loose since it's a working DM document, but stay narrative.

Read by every agent on every turn (surfaced in TURN_START).

For the design behind this, see [`../../docs/design/context-packages.md`](../../docs/design/context-packages.md).
