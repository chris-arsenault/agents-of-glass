# Research

This directory holds:

1. **Summaries of the supporting repos** that inform this project but are not part of it.
2. **Placeholders for future research** — investigations we'll want to do but haven't yet.

The notes here are working documents. They get updated when we re-read source material, when we test an assumption against the actual repo, or when a new question becomes relevant.

## Existing Notes

- [the-glass-frontier-lore.md](the-glass-frontier-lore.md) — the worldbuilding wiki + graph database project that supplies our setting and lore patterns
- [the-glass-frontier.md](the-glass-frontier.md) — the TypeScript chronicle engine that supplies our game-design pieces (skill checks, intent taxonomy, beat tracking)

## Placeholders for Future Research

These are topics we expect to investigate but haven't started. Each placeholder is one paragraph stating *what* we want to know and *why* we'd want to know it.

- [ice-remembers.md](ice-remembers.md) — the narrative-weaving project we want to feed transcripts into
- [transcript-analysis-techniques.md](transcript-analysis-techniques.md) — what kinds of analysis are tractable on TTRPG transcripts
- [agent-personality-stability.md](agent-personality-stability.md) — keeping the five people consistent across many invocations
- [closure-signals-in-fiction.md](closure-signals-in-fiction.md) — what marks a scene "done" in human-written fiction, for cribbing into the closer agent
- [graph-coherence-checks.md](graph-coherence-checks.md) — what consistency checks the lore repo runs, and which we'd want

## Conventions

- Each summary doc has: **what it is**, **what we're cribbing**, **what we're explicitly not cribbing**, and **open questions for us**.
- Placeholders are short — one paragraph stating the question and why it matters. Fill them in when we actually do the research.
- Cross-link liberally. Research notes are most useful when they point at the design docs they justify.
