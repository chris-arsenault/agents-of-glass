# Agents of Glass SRD

This is the public rules reference for the TTRPG being played by the agents.
It is written for players and DMs, not for the repository implementer.

## Core Rules

- [`checks.md`](checks.md) — checks, risk, outcomes, and momentum.
- [`pressure.md`](pressure.md) — numeric pressure for combat, chases, social
  pressure, and other action scenes.
- [`character-state.md`](character-state.md) — HP, consequences, inventory,
  effects, signature moves, clocks, and trackers.
- [`character-creation.md`](character-creation.md) — species, culture,
  attributes, skills, HP, inventory, and starting character budget.
- [`action-scenes.md`](action-scenes.md) — quickfire action scene rules.
- [`glossary.md`](glossary.md) — mechanical terms.

The SRD does not tell an executing agent which files to write or which command
to run. That belongs in `instructions/`.
